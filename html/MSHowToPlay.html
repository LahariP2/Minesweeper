MineSweeper: How To Play

MineSweeper is a fun and challenging game in which the player has to try to clear the entire board without revealing any hidden mines. Once all the cells, which are not mines, are revealed, the player wins! The player also wins if all the mines are flagged! There are infinite number of ways a player can win a game using both luck and skill!

Types of Cells:

   Blank cell: non-revealed cell

   Mine

   Neighbors: When a blank cell is clicked, if it is not a mine, the number of adjacent cells containing mines will be displayed. The minimum number of neighboring mines is 0 and the maximum is 8.

   Flag   ~ If you think that a non-revealed cell is a mine, you can flag it by right clicking
